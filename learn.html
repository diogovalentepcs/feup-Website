First Steps On The Web

Welcome to the world of web criation!

You are on you're way to learn the fundamentals of page building! Alright, first of all you need to know how to speak "web". There are 3 core languages: HTML, CSS and JavaScript, which correspond to the skeleton, appearence and the brain of a web page.

The first thing you need to learn is HTML. Why? HTML is often the first language learned by marketers and designers and is core to front-end (layout) development work. If this is your first time touching code, I am excited for what you're about to create.

Learn a litle bit more about these languages, the frameworks and the best platforms to code down below!

Let's start!

Languages

HTML

HTML provides structure to the content appearing on a website, such as images, text, or videos. Right-click on any page on the internet, choose “Inspect,” and you’ll see HTML in a panel of your screen.

  • Learn More

    • CSS

    Cascading Style Sheets (emphasis placed on “Style”) comes to specifiy your document’s style—page layouts, colors, and fonts. Think of HTML as the foundation (every house has one), and CSS as the aesthetic choices (there’s a big difference between a Victorian mansion and a mid-century modern home).

  • Learn More

    • JS

    Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web. JavaScript enables interactive web pages and thus is an essential part of web applications. The vast majority of websites use it, and all major web browsers have a dedicated JavaScript engine to execute it.

  • Learn More

    • Frameworks


    Web frameworks have transformed the world of programming! A framework simply automates the coding process by giving us CSS and JS code "shortcuts" that are already formatted but totally changeable. I used Bootstrap (my favourite) for this webite, just download the framework and start coding with the shortcuts!
